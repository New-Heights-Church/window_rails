WindowRails

WindowRails is a plugin for Rails that provides easy to use AJAXy windows. It is based completely on Prototype Window with helpers for Rails to make it easy to use from a Rails app.

Configuration

After install, add these in the head of your application.html.erb file: 

<%= javascript_include_tag 'window_rails/window' %>
<%= stylesheet_link_tag *%w(/themes/alphacube /themes/default themes/alert) %>

Alphacube is the default theme used by window_rails. Be sure to add the css link if you plan on using other themes.

Basic examples to get you started

Linking to a window

<%= link_to_window('Link', :url => {:controller => :my_controller, :action => :my_action}, :modal => true, :className => 'alphacube', :window => 'some_name') %>

Opening a window via AJAX

def open_my_window
  respond_to do |format|
    format.html
    format.js do
      render :update do |page|
        page.open_window({:partial => 'my_partial'}, :modal => true, :width => 400, :height => 200, :window => 'my_window')
      end
    end
  end
end

Updating a window via AJAX

def update_my_window
  respond_to do |format|
    format.html
    format.js do
      render :update do |page|
        page.update_window({:partial => 'new_partial'}, :window => 'my_window')
      end
    end
  end
end

Closing a window via AJAX

def close_my_window
  respond_to do |format|
    format.html
    format.js do
      render :update do |page|
        page.close_window(:window => 'my_window')
      end
    end
  end
end
